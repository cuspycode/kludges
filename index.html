BD's Fulhack

"Fulhack" is the swedish word for "very elegant small piece of program".

Here are some code samples from me that I hope qualify for the prestigious fulhack label.

  • UTC in crontabs - Yes it's possible. Click here to free your crontabs from the DST madness.
  • Recording video on a Linux Xbox - I've done some preliminary experiments with a Hauppauge WinTV-PVR-usb2.
  • GLOTV - Gnu Lightweight Opuscule TeleVision.
  • JuestBook - Java guestbook that lets you paint and upload GIF pictures.
  • ISO 8601 week numbers - Sample Perl code to calculate week numbers (veckonummer).
  • buffer - A simple C program that buffers a data stream.
  • skyffla.pl - A file transfer program kludge thing that I use all the time, for some reason.
  • NNTP via multicast in Perl - It works but who cares? It's only bandwidth...
  • A tftp replacement that uses TCP - You don't want to know why :)
  • proxy3 - My very first (or 3rd rather) homebrew socket forwarding program in C.
  • MAC address collector - My first Perl program using raw sockets.

I'm not done yet... more crazy bits and bytes coming up soon! ;)
